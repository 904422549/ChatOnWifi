# ChatOnWifi
Android平台局域网聊天工具
 
 
# 通过使用自定义协议，将所有用户上下线信息，消息，文件传输请求等命令封装为自定义协议数据包，使整个框架层次清晰，也方便以后功能的拓展。思想借鉴与飞鸽传书软件。
